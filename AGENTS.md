# AI 协作入口

本仓库是 TypeScript 前端水印库，默认使用中文沟通、中文规则和 pnpm 命令。

开始工作前，按以下顺序阅读：

1. [.agents/project-map.md](.agents/project-map.md)：结构、边界与产物真源。
2. [.agents/decision-gates.md](.agents/decision-gates.md)：是否可直接执行的判断标准。
3. 与任务对应的 `.agents` 文件：命令、模块规范或验证规则。

不要复制 `.agents` 中的细则到本文件。面向使用者的安装和 API 用法以 [README.md](README.md) 为准；面向人类的长期背景与决策才放入 `docs/`。

提交、推送、发布，以及任何未通过决策门槛的修改，必须先获得明确确认。
