# API 说明

```ts
import { watermark } from 'common-watermark';
```

## `watermark.setWatermark(content, element)`

为指定容器创建水印。

| 参数      | 类型          | 说明                                 |
| --------- | ------------- | ------------------------------------ |
| `content` | `string`      | 水印文本；逗号或换行符会分隔为多行。 |
| `element` | `HTMLElement` | 需要覆盖水印的目标容器。             |

同一个 `element` 再次调用时，旧水印会先被清理，再按新文本创建。不同容器可以同时设置水印。

## `watermark.removeWatermark(element)`

移除目标容器上的水印，并停止该实例的观察器、事件监听器和计时器。

| 参数      | 类型          | 说明                   |
| --------- | ------------- | ---------------------- |
| `element` | `HTMLElement` | 已设置水印的目标容器。 |

## 行为说明

- 覆盖层不拦截指针事件。
- 当目标元素处于静态定位时，库会临时建立定位上下文，并在销毁时恢复原有内联定位值。
- 容器本身或其内容发生尺寸变化时，水印覆盖范围会自动更新。
